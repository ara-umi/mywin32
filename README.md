# 个人用后台键鼠模拟库mywin32
## 目标

​	通过win32系列库发送足够真实的后台键鼠消息，从而实现对指定窗口的后台模拟。

​	主要应用场景：安卓模拟器上运行的各类脚本。

## 功能

​	寻找模拟器句柄（handle.py，兼容性未测试）

​	            目前用于测试的模拟器有：Nox和LeiDian

​	键鼠模拟（类似后台版的pyaugui，或低配版的大漠插件）
